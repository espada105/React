JavaScript Array Methods - 실습 테스트

1. map() - 제곱 배열

문제: 배열 [1, 2, 3]의 각 요소를 제곱하여 새로운 배열을 만드시오.

const input = [1, 2, 3];
const result = input.map(x => x * x);
console.log(result); // [1, 4, 9]

2. filter() - 짝수만

문제: 배열 [1, 2, 3, 4, 5, 6]에서 짝수만 골라 새 배열을 만드시오.

const input = [1, 2, 3, 4, 5, 6];
const result = input.filter(x => x % 2 === 0);
console.log(result); // [2, 4, 6]

3. find() - 10보다 큰 첫 값

문제: 배열 [3, 7, 11, 15]에서 10보다 큰 첫 번째 값을 구하시오.

const input = [3, 7, 11, 15];
const result = input.find(x => x > 10);
console.log(result); // 11

4. push() & pop() - 스택 동작

문제: 빈 배열에 10, 20, 30을 순서대로 넣고 마지막 값을 제거한 후 배열을 출력하시오.

const stack = [];
stack.push(10);
stack.push(20);
stack.push(30);
stack.pop();
console.log(stack); // [10, 20]

5. reduce() - 합계

문제: 배열 [5, 10, 15]의 합계를 구하시오.

const input = [5, 10, 15];
const result = input.reduce((acc, cur) => acc + cur, 0);
console.log(result); // 30

6. sort() - 오름차순 정렬

문제: 배열 [7, 2, 10, 1]을 오름차순으로 정렬하시오.

const input = [7, 2, 10, 1];
const result = input.sort((a, b) => a - b);
console.log(result); // [1, 2, 7, 10]

7. includes() - 포함 여부

문제: 배열 ["apple", "banana", "cherry"]에 "banana"가 포함되어 있는지 확인하시오.

const input = ["apple", "banana", "cherry"];
const result = input.includes("banana");
console.log(result); // true

8. join() - 문자열로 결합

문제: 배열 ["apple", "banana", "cherry"]를 쉼표와 공백으로 연결된 문자열로 만드시오.

const input = ["apple", "banana", "cherry"];
const result = input.join(", ");
console.log(result); // "apple, banana, cherry"

9. slice() - 부분 배열 추출

문제: 배열 [10, 20, 30, 40, 50]에서 인덱스 1부터 3까지(3은 제외) 잘라낸 배열을 만드시오.

const input = [10, 20, 30, 40, 50];
const result = input.slice(1, 3);
console.log(result); // [20, 30]
